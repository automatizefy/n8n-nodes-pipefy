# Contexto do Produto

## Por que este projeto existe?
O n8n é uma ferramenta poderosa de automação de fluxo de trabalho, e o Pipefy é uma plataforma líder em gerenciamento de processos de negócios. Este nó foi criado para permitir que usuários do n8n integrem facilmente seus fluxos de trabalho com o Pipefy, automatizando tarefas e processos.

## Problemas que resolve
1. **Complexidade da API**: A API do Pipefy é baseada em GraphQL, que pode ser complexa para usuários não técnicos. Este nó simplifica essa interação.
2. **Integração Manual**: Sem este nó, usuários precisariam criar integrações personalizadas usando HTTP requests.
3. **Validação de Dados**: O nó fornece validação automática de dados e tratamento de erros.
4. **Campos Personalizados**: Gerencia automaticamente diferentes tipos de campos do Pipefy.

## Como deve funcionar
1. **Autenticação Simples**: Usuários fornecem apenas o token de API do Pipefy.
2. **Interface Intuitiva**: Todas as operações são apresentadas de forma clara e organizada.
3. **Validação Automática**: Os dados são validados antes de serem enviados ao Pipefy.
4. **Feedback Claro**: Mensagens de erro e sucesso são claras e informativas.

## Objetivos de Experiência do Usuário
1. **Simplicidade**: Usuários devem conseguir configurar o nó em poucos minutos.
2. **Clareza**: Cada operação e campo deve ter descrições claras.
3. **Confiabilidade**: O nó deve lidar graciosamente com erros e fornecer feedback útil.
4. **Flexibilidade**: Suportar todos os tipos de campos e operações do Pipefy.

## Integrações Principais
1. **Cards**: Gerenciamento completo de cards, incluindo campos personalizados.
2. **Pipes**: Criação e gerenciamento de pipes e suas configurações.
3. **Phases**: Controle de fases e seus campos específicos.
4. **Organizations**: Gerenciamento de organizações e membros.
5. **Tables**: Suporte para tabelas e registros de tabela.
6. **Webhooks**: Configuração de webhooks para eventos do Pipefy.
